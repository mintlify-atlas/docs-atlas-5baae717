## What is FluidCalendar?

FluidCalendar combines smart automatic task scheduling with a full-featured calendar, helping you stay on top of your tasks without manual planning. It finds optimal time slots for your tasks based on your work hours, energy levels, and existing calendar events — so you spend less time managing your schedule and more time executing.

<Note>
FluidCalendar is in active development. You may encounter bugs. Please [report issues](https://github.com/dotnetfactory/fluid-calendar/issues) to help improve the project.
</Note>

## Open source vs SaaS

FluidCalendar is available as both a self-hosted open source application and a managed SaaS service.

| Feature | Open Source | SaaS |
|---|---|---|
| Calendar management | ✓ | ✓ |
| Task management | ✓ | ✓ |
| Google Calendar sync | ✓ | ✓ |
| Outlook Calendar sync | ✓ | ✓ |
| CalDAV integration | ✓ | ✓ |
| Billing & subscriptions | — | ✓ |
| Managed infrastructure | — | ✓ |
| Automatic updates | — | ✓ |

## Tech stack

FluidCalendar is built with:

- **Next.js 15** with App Router
- **TypeScript** throughout
- **Prisma** ORM with PostgreSQL
- **FullCalendar** for the calendar UI
- **NextAuth.js** for authentication
- **Tailwind CSS** for styling
- **BullMQ + Redis** for background jobs (SaaS)
